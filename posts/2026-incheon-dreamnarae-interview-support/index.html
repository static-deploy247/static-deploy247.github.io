2026 인천 드림나래 신청안내 (면접정장·컨설팅 지원)

2026-03-02

한눈에 결론

인천 드림나래는 면접 정장 대여와 이미지 컨설팅을 함께 준비할 수 있는 청년 지원 프로그램입니다.
신청 전에는 제출서류의 핵심 3요소(회사명·본인이름·면접 일정)와 예약·반납 기준을 먼저 맞추는 것이 가장 중요합니다.

지원 대상

  • 채용 목적 면접을 앞둔 구직 청년
  • 재직자도 이직 면접 예정이면 신청 가능
  • 진학·모의면접·교육과정 실습 등 채용 목적이 아닌 면접은 지원 제외

인천 면접 정장 대여 무료 지원 내용

  • 면접용 정장세트: 셔츠/블라우스 + 바지/치마 + 자켓 + 넥타이
  • 구두 포함 대여
  • 대여 기간: 3박 4일(업체 운영일 기준)
  • 반납 지연 시 1일 1만원 수준의 연체 수수료가 발생할 수 있음

신청 절차

  • 온라인 신청
  • 승인 확인
  • 배정된 제휴업체 방문 대여
  • 3박 4일 내 반납

면접 일정이 대여 기간 안에 포함되도록 날짜를 잡는 것이 안전합니다.

대여 장소 확인 방법

  • 승인 후 배정된 제휴 정장업체 정보(지점/주소) 확인
  • 업체별 운영일, 연장 가능 여부가 달라 방문 전 사전 확인

제출서류 체크리스트

  • 인천 거주 확인 서류: 주소지 조회 또는 관내 대학 재학·휴학증명
  • 면접 예정 확인 자료: 문자/메일/공고문/수험표/통화기록 중 해당 자료
  • 공통 확인 항목: 회사명 + 본인이름 + 면접 일정

면접 예정 확인 자료 인정 기준

  • 공통 원칙: 회사가 확인되는 공식 자료여야 함
  • 문자 안내: 발신번호가 회사 대표번호 또는 채용 담당 번호인지 확인
  • 메일 안내: 발신 메일 도메인(@ 뒤)이 회사 메일인지 확인
  • 공고문/홈페이지: 회사 로고·URL 등 식별 가능 정보 필요
  • 수험표/응시표: 회사 식별 정보와 일정 확인 가능해야 함
  • 통화기록: 통화내역 캡처 + 지원내역을 함께 제출

개인정보(주민등록번호, 사진 등)는 블라인드 처리 후 제출하는 것이 권장됩니다.

이미지 컨설팅 대상 및 준비자료

  • 구직 상태를 증명할 수 있는 청년은 신청 가능
  • 면접 일정이 아직 없어도 신청 가능
  • 진행 내용: 1:1 퍼스널 컬러 진단 + 면접 자세 코칭(30~40분)

필요 서류는 거주 확인 + 구직활동 확인 자료(고용24 구직등록확인증, 채용포털 구직활동확인서 등)입니다.

예약·취소 유의사항

  • 예약/변경/취소는 예약일 1일 전 17시까지
  • 당일 변경·취소는 횟수 차감 가능
  • 컨설팅과 정장 대여는 동시 예약 가능

신청 전 최종 점검

  • 면접이 채용 목적 일정인지 확인
  • 서류에 회사명·본인이름·면접 일정 3요소가 모두 보이는지 확인
  • 반납일과 연체 가능성을 고려해 대여일 선택
